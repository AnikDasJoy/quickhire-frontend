# QuickHire – Job Portal Web Application

QuickHire is a full-stack job portal web application where users can search for jobs, view job details, and apply for positions, while administrators can manage job postings.

This project is built using the MERN stack with Firebase Authentication.

---

## Features

### User Features
- User registration and login (Firebase Authentication)
- Browse available jobs
- Search jobs by keyword
- Filter jobs by location and category
- View job details
- Apply for jobs

### Admin Features
- Secure admin dashboard
- Add new jobs
- Manage job listings
- View job applications
